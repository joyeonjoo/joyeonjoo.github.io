기술 스택

프로젝트 소개
초등 교육 관련 상품을 판매하는 쇼핑몰을 Angular 기반으로 구축하고, 공통 UI 컴포넌트를 설계하여 유지보수성과 재사용성을 높였습니다.

담당 업무

  • Angular Template 작성 및 TypeScript 수정
  • 재사용 가능한 공통 UI 컴포넌트 설계 및 제작
  • SCSS 변수 및 mixin 기반 스타일 시스템 구축
  • Angular 라우팅 및 컴포넌트 구조 개선
  • Git 협업 및 코드 리뷰 참여
  • 반응형 웹 구현 및 웹 접근성 고려한 마크업 작성

주요 화면

어려웠던 점

기존에 공통 컴포넌트가 없어 동일한 UI가 여러 페이지에 중복되어 유지보수가 어려웠고, 스타일 관리가 일관되지 않아 디자인 수정 시 많은 시간이 소요되었습니다.

해결 방법

공통 UI 컴포넌트를 설계하여 재사용 가능하도록 구조화하고, SCSS 변수와 mixin을 활용해 스타일 시스템을 구축하여 유지보수 효율을 향상시켰습니다.

결과 및 배운 점

코드 중복이 줄어 유지보수가 쉬워졌고, 개발자와의 협업 효율이 높아졌습니다. 컴포넌트 기반 설계의 중요성과 체계적인 스타일 관리의 필요성을 배웠습니다.
